# PCB

KiCad project goes here: schematic (`.kicad_sch`), PCB layout (`.kicad_pcb`), project file (`.kicad_pro`), and exported Gerbers/drill files.

Reference: [Designing the PCB](https://keeb.hackclub.com/docs/pcb-design) and [marbastlib](https://github.com/ebastler/marbastlib) for keyboard-specific footprints (switches, stabilizers).

Key components: `RaspberryPi_Pico`, `SW_Push` (×68), `MX_stab` (×~5), `1N4148` (×68), 0.91" OLED, EC11 encoder, SK6812 MINI-E.
