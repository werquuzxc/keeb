# Case

Onshape exports for the case and plate go here: `.step` (source/remix) and `.stl` (for slicing/printing).

Reference: [Designing the Case](https://keeb.hackclub.com/docs/case-design). Built around the PCB's exact dimensions (imported as STEP from KiCad), rectangular shape with a slight typing angle, switch holes at 14×14mm / 19.05mm spacing, USB-C cutout, M3 heatset insert bosses.

3D-printer bed is typically ~220×220mm — a 65% board may need the case split into two halves with alignment pins if it doesn't fit in one piece.
